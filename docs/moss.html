Moss features:

  • Start a new group with a click, no agreements or creditcards reqiured.
  • Large suit of possible tools available
  • Make or edit a tool to fit with the needs of your group

Desktop application

Moss is currently a desktop application that is available for all most operating systems (Mac OS / Windows / Linux).

Once you have installed it you can create a group, install some tools into the group and start inviting others to join.

If you have recieved an invitation, all you need to do is to install the application and click the "join group" button where you paste the link you have been sent.

We you join a group, all of the tools the group is using are waiting for you to use there, you just install the ones you wish to work with.

If you would like to play with another tool with someone in the group you can install that tool from the tool library and it will show up as beeing in use for the others and they can join that one too.

Let your groups grow, or shrink over time in both members and complexity. Moss gives your group the ability to have your collaborations evolve and change along with the group itself!
